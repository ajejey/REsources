Products


Vayumithra

HC3-D

Horizontal Continuous Clean Combustion Device


The VAYUMITRA is a continuous combustion device tailored to cater the need of domestic to large scale cooking application. Suitable for domestic application, catering units, hotels, restaurants, and canteens.


Models: 1 kg/hr, 4 kg/hr, and 6 kg/hr

Features

  • Uses state of the art technology for ensuring clean combustion
  • Variable flame control
  • Fuel Flexibility
  • Easy Ash collection system
  • Uninterrupted operation
  • Smokelesss clean cooking experience
  • No soot formation
  • Maximum firewood size of 15-20cm

Agnisakhi

VEBCOD

Vertical Ejector Based Combustion Device


The AGNISAKHI is a clean combustion device uniquely designed for the easy replacement solution for the traditional cook stove systems, in which long logs of wood can be used (limited to window size). Suitable for domestic application, catering units, hotels, restaurants, and canteens


Models: 1 kg/hr, 4 kg/hr, and 6 kg/hr

Features

  • Uses state of the art technology for ensuring clean combustion
  • Variable flame control
  • Fuel Flexibility
  • Easy Ash collection system
  • Uninterrupted operation
  • Smokelesss clean cooking experience
  • No soot formation
  • Long wood with width limited to window size can be used
